# Редактор запросов 1С (QueryEditor1C)

Десктопный редактор для редактирования запросов 1С:Предприятие с подсветкой синтаксиса.

## 🚀 Возможности

- ✅ **Подсветка синтаксиса** — поддержка всех ключевых слов и функций 1С
- ✅ **Редактирование** — полноценный текстовый редактор на базе AvalonEdit
- ✅ **Файловые операции** — создавайте, открывайте и сохраняйте запросы
- ✅ **Нумерация строк** — удобная навигация по коду
- ✅ **Темы оформления** — светлая и темная темы
- ✅ **Статусная строка** — отображение позиции курсора
- ✅ **Отмена/повтор** — поддержка истории изменений

## 📋 Системные требования

- Windows 10/11
- [.NET SDK 10.0](https://dotnet.microsoft.com/download) или выше

## 🛠️ Сборка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd queryeditor

# Сборка проекта
dotnet build

# Запуск
dotnet run
```

Или запустите скомпилированный файл:
```
bin\Debug\net10.0-windows\QueryEditor1C.exe
```

## 📝 Использование

1. **Создание нового файла**: Файл → Новый (Ctrl+N)
2. **Открытие файла**: Файл → Открыть (Ctrl+O)
3. **Сохранение**: Файл → Сохранить (Ctrl+S)
4. **Форматирование**: Формат → Форматировать запрос
5. **Смена темы**: Выберите тему в выпадающем списке на панели инструментов

`ВЫБРАТЬ`, `ИЗ`, `ГДЕ`, `СОЕДИНЕНИЕ`, `ЛЕВОЕ`, `ПРАВОЕ`, `ВНУТРЕННЕЕ`, `ПО`, `И`, `ИЛИ`, `НЕ`, `ПЕРВЫЕ`, `РАЗЛИЧНЫЕ`, `КАК`, `ПОМЕСТИТЬ`, `УПОРЯДОЧИТЬ`, `СГРУППИРОВАТЬ`, `ИМЕЮЩИЕ`, `ОБЪЕДИНИТЬ`, `ВСЕ` и многие другие.

### Функции
`ЕСТЬNULL`, `ПОДСТРОКА`, `ДАТА`, `НАЧАЛОПЕРИОДА`, `КОНЕЦПЕРИОДА`, `ДОБАВИТЬКДАТЕ`, `РАЗНОСТЬДАТ`, `СУММА`, `МАКСИМУМ`, `МИНИМУМ`, `КОЛИЧЕСТВО`, `СРЕДНЕЕ` и др.

## 🔧 Технологии

- **WPF** (.NET 10) — фреймворк пользовательского интерфейса
- **AvalonEdit** — компонент редактирования кода с подсветкой синтаксиса
- **XSHD** — XML-определение правил подсветки синтаксиса
